 = 실행중인 프로그램

쓰레드
 = 실행될 명령어들의 연속
 = 단일 쓰레드 프로그램
   - 하나의 실로 꿸 수 있는 프로그램
 = 다중 쓰레드 프로그램
   - 프로그램에서 여러개의 함수들이 서로 독립적으로 수행
   - 이런 경우에는 명령어들이 순차적으로 수행되지 않기 때문에 하나의 실로 꿸 수 없다.

자바의 쓰레드  // C에서는 다중 상속이 되지만, JAVA에서는 다중 상속 불가능(extend)
 = Thread 클래스로부터 상속받아서 사용			// 클래스와는 무관하게 독립된 작업을 하거나 여러개의 쓰레드가 필요할 경우
 = Runnable 인터페이스를 Inplements해서 사용		// 클래스의 특성을 그대로 유지하면서 쓰레드를 하나 더 생성할 수 있음

 = 두 방법 모두 쓰레드 사용하려면 run()함수 만들어야함
  - run()이 직접적으로 실행될 내용
  - 시작 = start()
  - start()가 호출되면 자동적으로 run()
 
 = run()을 마치거나 stop()을 하면 dead상태로 전이
  - dead일 때는 모든 작업이 종료됨, 다시 start()로 실행 불가

 = stop(), resume(),suspend()는 jdk 1.2에서 워닝 뜸.
